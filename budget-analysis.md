> Дата: 2026-02-26. Цены из `pricing.html`, расходы Mid.

---

## Ключевое решение

Артём убрал из бюджета организатора:
- **Ужины** — каждый платит сам
- **Шоу** (Alizée, Friedrichstadt-Palast) — каждый покупает билет сам
- **Фасилитаторы и ведущий** — волонтёры / без оплаты
- **Транспорт** (автобусы) — не нужны без групповых ужинов

Экономия: **~€37K** при 70 участниках.

### Чувствительность к миксу Creative / Tech

Техи платят €690–1190, креативы €290–790. Микс сильно влияет на break-even.

| Микс (при 70 чел, 1P+1S) | Результат |
|---------------------------|----------:|
| 40% creative / 60% tech | +€7,598 |
| 50% / 50% | +€4,958 |
| **60% / 40% (базовый)** | **+€2,318** |
| 70% creative / 30% tech | −€322 |

Вывод: чем больше tech-участников — тем быстрее break-even. При 70% креативов (мало техов) ивент на 70 чел уже не окупается.

## Рычаги роста прибыли

Каждый дополнительный партнёр — почти чистая прибыль (расходы растут только на комиссию 4%):
- +1 Partner = **+€2,880 net**
- +1 Strategic = **+€7,680 net**
- +1 Title = **+€14,400 net**

Каждые дополнительные 10 участников (сверх break-even) ≈ **+€5K–7K net**.
